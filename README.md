# Electron Shopping Cart

An electron shopping cart app that uses materialize.css for styling

- Electron v1.7.6
- materialize.css v0.100.2 

## Basic Usage

```shell
npm install

# set environment to dev (inside main.js)
process.env.NODE_ENV = 'dev' 

# run the app
npm start

# build for mac
package-mac

# build for windows
package-win

# build for linux
package-linux
```
